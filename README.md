# OpenDS

OpenDS is an open-source, self-hosted design system platform that bridges design tools with development workflows.

## Development
```bash
npm i
npm run dev
```

## Workspaces
- `frontend/` — Vue 3 app
- `backend/` — Fastify API

## Testing
- Unit: Vitest (`npm run test`)
- E2E: Playwright (`npm run e2e` in `frontend/`)

## Docker
```bash
npm run docker:up
```
